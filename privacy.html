Privacy Policy

Last updated: 2026-06-14

1. Overview

This application ("Animal TikTok Bot") is a personal automation tool that generates AI animal videos and posts them to TikTok. It is operated by a single individual for personal, non-commercial use.

2. Information We Collect

This application accesses the following data through the TikTok API, solely for the purpose of posting videos:

  • TikTok OAuth access token (stored locally on the operator's device only)
  • Creator account information required by the Content Posting API

No personal data of any third party is collected, stored, or shared.

3. How We Use Information

The OAuth token is used exclusively to authenticate API requests for video publishing. It is stored in a local file on the operator's machine and is never transmitted to any third party.

4. Data Sharing

We do not sell, share, or disclose any data to third parties. All data remains on the local device of the application operator.

5. Data Retention

Access tokens are stored locally and can be deleted at any time by removing the .tiktok_token.json file. Generated videos are deleted automatically after upload.

6. Third-Party Services

This application uses the following external APIs:

  • TikTok Content Posting API – subject to TikTok's Privacy Policy
  • Stability AI API – subject to Stability AI's Privacy Policy
